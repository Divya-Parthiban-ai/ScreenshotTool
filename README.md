# Screenshot to Word Tool

## Features
- Capture screenshots using Ctrl + Alt + S
- Auto insert into Word document
- Step description + Pass/Fail
- Organized per use case folder

## Setup

1. Install Python
2. Install dependencies:
   python -m pip install pyautogui python-docx keyboard pillow

## Run

python screenshot_to_word.py

## Usage
- Enter Use Case Name
- Press Ctrl + Alt + S for each step

## Important
- Do NOT open Word file while capturing
